# Data Structures &amp; Algorithms in C++

## Dependency

```text
cmake >= 3.13 
gcc >= 9.3.0
```

### To compile

1. Create build directory with

```bash
mkdir -p -v build
```

2. Enter the build directory and run cmake

```bash
cd build; cmake ..
```

3. To run a particular executable

```bash
./arrays/1_setMatrixZeros
```
